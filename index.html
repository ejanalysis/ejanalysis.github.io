EJ Analysis on GitHub

Data and R code for environmental justice (EJ) analysis

About

These GitHub repositories provide data and code (in R) for analyzing environmental justice (EJ) data.

Packages

  • countyhealthrankings - Health indicators by US County
  • UScensus2010blocks - 11 million Census blocks
  • analyze.stuff - General tools for data analysis
  • proxistat - Proximity/distance analysis, and block group data
  • ejanalysis - Environmental justice/ Demographic/ Environmental metrics
  • ACSdownload - American Community Survey 5-year summary file -- download a table for all US Census tracts or block groups
